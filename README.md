# 日程クリッパー（Chrome拡張）

日程候補の提案文（「以下の日程でご都合いかがでしょうか。」+ 候補日時の一覧）を、
ツールバーのポップアップから素早く作成・コピーできるChrome拡張機能。

© Ryota Kashiwagi

## 使い方

1. カレンダーで日付をクリック（複数可）
2. 必要なら開始時間・終了時間を選択（日付のみでもOK）
3. プレビューの提案文を「コピーする」でクリップボードへ

選択状態はポップアップを閉じても保持されます（ブラウザ再起動またはリセットで消去）。

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択

## 構成

- `manifest.json` … Manifest V3（権限: clipboardWrite / storage）
- `popup.html` / `popup.css` / `popup.js` … ポップアップ本体（ビルド不要）
- `icons/` … アイコン
- `docs/` … 設計メモ
