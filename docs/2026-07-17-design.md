# 日程クリッパー 設計メモ

作成日: 2026-07-17

## 目的

日程調整のたびに候補日時を手打ちするのをやめ、Chrome拡張のポップアップから
数クリックで提案文を作成・コピーできるようにする。

## 構成

- Manifest V3 / ビルド不要のプレーン HTML・CSS・JS
- `manifest.json` … 権限は `clipboardWrite` + `storage` のみ
- `popup.html` / `popup.css` / `popup.js` … 幅380pxの縦1カラム
- `icons/` … 16/48/128px

## 機能

1. カレンダー（月送り、日曜・祝日=赤、土曜=青、過去日は選択不可）
2. 日付クリック → スロット追加 → 開始時間グリッド（8:00〜22:30、30分刻み）
3. 開始選択 → 終了時間グリッド → 選択で確定（日付のみ・開始のみでも可）
4. プレビュー: 「以下の日程でご都合いかがでしょうか。」+ 日時ソート済み候補行
   - 形式: `7月20日（月）　10:00〜11:30`（全角スペース区切り、時刻は `H:MM`）
5. コピー / リセット
6. 祝日データは 2025〜2027 の静的リストを内蔵

## ポップアップ向けの設計判断

- 選択状態を `chrome.storage.session` に保存（ポップアップが閉じても選択が消えない。
  リセット・ブラウザ再起動で消える）。`chrome` API がない環境（通常ブラウザで
  popup.html を直接開いた場合）でも動くようフィーチャーディテクトする。
