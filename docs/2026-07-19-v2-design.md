# 日程クリッパー v2.0.0 設計メモ

作成日: 2026-07-19

## 目的

公開（Chrome Web Store等）を前提に、コード・デザイン・機能の3層を独自化した全面リライト。

## アーキテクチャ

- **状態駆動**: 単一の `state` オブジェクト + `render()`。
  イベント → `update(mutate)` → 永続化 → 全体再描画 の一方向フロー。
- **祝日計算エンジン** (`holidays.js`): 静的リストを持たず、祝日法ルールから導出。
  - 固定祝日 / ハッピーマンデー（第n月曜） / 春分・秋分（1980〜2099年有効の天文近似式）
  - 振替休日（日曜の祝日 → 次の非祝日） / 国民の休日（祝日に挟まれた平日）
  - `test/holidays.test.js` で内閣府公表の2025〜2027年一覧と突合済み
- **出力形式** (`formats.js`): 候補1件を1行に整形する関数として形式を定義。
  標準／箇条書き／English の3種。追加も1エントリで可能。
  書き出し文（挨拶文）は付けず、候補日時の行のみを出力する。

## UI・インタラクション

- 時間帯は**1つのグリッドで範囲選択**（1回目=開始、2回目=終了。
  開始以前をタップで選び直し）。「時間指定しない」「完了」も用意。
- 候補はリスト表示し、**個別に削除・時間の再編集**が可能。
- 出力形式はセグメントコントロールで事前選択、`chrome.storage.sync` に保存。
- 候補・表示月・編集状態は `chrome.storage.session` に保存。

## デザイン

- 文具モチーフ「クリーン&モダン」: 紙白×墨×インク青緑（#0E7A6E）の1アクセント。
  日曜・祝日の赤、土曜の青と干渉しない配色。
- シグネチャ: 生成文プレビューを**クリップで留めたメモ**として表現
  （ペーパークリップSVG + わずかな傾き）。
- ダークモード（prefers-color-scheme）、reduced-motion、focus-visible対応。
- 数字はタブラー数字（font-feature-settings: 'tnum'）で桁揃え。
